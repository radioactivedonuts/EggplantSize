# 🍆 How Eggplant Are You?

A ridiculously fun single-file luck tester with maximum eggplant size.

## Updated to have better odds tag me on socials @Eggplants4Lyfe 

## 🎮 How to Play

1. Click the big purple button **"TEST YOUR EGGPLANT"**
2. Discover how legendary (or tragic) your eggplant luck is today
3. Repeat until you get the ultimate 🍆 result

## ✨ Features

- **Zero bloat** — Just one beautiful HTML file
- Pure eggplant-themed luck testing
- Dramatic reactions based on your roll
- Beautiful purple gradient design
- Fully responsive (works great on mobile)
- No ads, no tracking, no nonsense
